YourPlatform

Verwaltungs- und Intranet-Plattform für geschlossene Anwendergruppen. Für Verbände, Unternehmen, Studenten- und Alumni-Netzwerke. Kommunikation, Information, Verwaltung und Networking. In einem Paket als freie Open-Source-Software.

Schnellsuche

Mit der Schnellsuche können Inhalte, Dokumente, Personen, Gruppen, Veranstaltungen, etc. gefunden werden.

Dokumente hochladen

Dokumente werden per Drag & Drop aus einem Ordner auf die entsprechende Seite gezogen, um sie dort hochzuladen. Die Lese-Rechte ergeben sich aus den Gruppenzugehörigkeiten, sodass Dokumente öffentlich, für begrenzte Zielgruppen, z.B. auch für Seminarteilnehmer etc. hochgeladen werden können.

Die Dokumente sind dann der entsprechenden Inhaltsseite als Anhang beigefügt.

Wenn man in die Zielgruppe eines Dokumentes fällt, wird einem das Dokument gemäß seiner Aktualität auf der Startseite angezeigt.

Anlegen eines Gremiums

Zur Demonstration legen wir eine Informationsseite für das Gremium "Senat" an. Die Informationsseite kann von allen angemeldeten Benutzern gelesen werden.

Als Untergruppe erstellen wir "Mitglieder des Senates".

Wir fügen der Gruppe zwei Mitglieder hinzu.

Der Gruppe fügen wir einen Ordner "Interne Senatsdokumente" hinzu, dessen Inhalte nur von Mitgliedern der Gruppe gelesen werden können.

Als globaler Administrator haben wir, um anderen Benutzern Hilfestellung zu geben, umfassende Berechtigungen und somit theoretisch auch die Möglichkeit, auf die Senatsdokumente zuzugreifen. Dieser Zugriff muss aber bestätigt werden und wird geloggt.

Sammelnachrichten

Leicht lassen sich Sammelnachrichten an alle Gruppenmitglieder schicken.

Diese Nachrichten lassen sich auch per E-Mail einspeisen. Hierzu können in den Einstellungen der Gruppe einfach entsprechende E-Mail-Verteiler definiert werden.

Wenn man danach eine Nachricht an eine solche Verteiler-Adresse schickt ...

... wird die Nachricht allen Gruppenmitgliedern als E-Mail zugestellt. Zudem wird die Nachricht im Nachrichtenarchiv der Gruppe geführt.

Die Nachricht wird den Gruppenmitgliedern auch auf ihren Startseiten angezeigt und kann dort z.B. kommentiert werden. Anhänge werden hochgeladen und entsprechend verlinkt.

Wer an die Verteiler senden darf, lässt sich in den Verteiler-Einstellungen der Gruppe festlegen.

Veranstaltungen eintragen

In der Gruppe "Konvent Erlangen-Nürnberg" tragen wir ein Konventstreffen als Veranstaltung ein.

In den Veranstaltungs-Details geben wir Beschreibung, Ort, Zeit und Ansprechpartner an. Außerdem wählen wir aus, ob die Veranstaltung im Feed für die öffentliche Homepage angezeigt werden soll.

Der Veranstaltung können Teilnehmer, Referenten und Veranstaltungsunterlagen hinzugefügt werden. Da die Teilnehmer auch eine Gruppe bilden, können damit auch Teilnehmerlisten exportiert oder Sammelnachrichten an Veranstaltungsteilnehmer geschickt werden.

Aktuelle Veranstaltungen, die einen betreffen, werden auf der Startseite angezeigt.

Die Termine, die einen betreffen, lassen sich per iCal-Feed mit dem eigenen Kalender auf Rechner, Tablet oder Smartphone verknüpfen, sodass dort relevante Termine automatisch angezeigt werden.

Amtsträger

Wenn wir den Konvent-Gruppen Amtsträger zuordnen ...

... werden diese beispielsweise auch auf der Übersichtsseite der Konvente angezeigt.

Die Amtsträger können auch interne Dokumente ablegen, auf die Amtsnachfolger dann auch Zugriff haben; außerdem können Amts-E-Mail-Adressen verwendet werden.

Kartenansicht

Suchergebnisse und Gruppen-Mitglieder können auf Karten angezeigt werden.

Hilfe-Gesuche

Hilfe-Gesuche können per Klick auf den "Hilfe"-Knopf im Kopfbereich schnell und einfach an das Team der Intranet-Betreuer abgesetzt werden.

Passwort ändern

Um bei der Wahl hinreichend sicherer Passwörter zu helfen, werden entsprechende Werkzeuge bereitgestellt.

E-Akte

Für die Verwaltung relevante Personen-Informationen können auf einer Übersichtsseite zusammengestellt angezeigt werden ("E-Akte").

Personen-Profile

Zum Austausch von Kontaktdaten können Profile auch von anderen Mitgliedern aufgerufen werden.

Den Profilen können beliebige Kontaktinformationen zugeordnet werden.

Informationen zu Studium und Beruf fördern das Networking.

Berechtigte Verwaltungs-Mitarbeiter können Status-Workflows auslösen.
